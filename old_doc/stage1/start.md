# 使用 ofa api

ofa api 从应用场景上主要分为 **$ api** 和 **封装功能**；

> **封装功能**属于 stage2-stage4 的内容；核心的组件封装、页面封装和应用渲染；

**$ api** 是stage1的内容，ofajs入门级的核心知识；

ofajs提供了大量类似 [jQuery](https://jquery.com/) 的方法，可以快速使 **$** 操作节点、修改元素属性和文本等基础功能；

如果你已有一定的web前端知识储备，接下来的学习会如鱼得水；
